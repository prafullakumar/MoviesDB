# MoviesDB
iOS App written in Swift 

**Flow:**

   ⎚ Fetch Latest Movie </br>
   ⎚ On Scroll down, fetches subsequent pages </br>
   ⎚ On Click, Shows Movie detail along with similer movies(if present) </br>
   
   
**Dependency:**
   Pod
   
**Lib:**
  Moya
  Alamofie
  Kingfisher
  
    
**includs:**
   XCTest
